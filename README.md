# Backend for Madrassati App
This is the backend for Madrassati App. It is built using Flask Framework.
# Getting Started
run the following :
```bash
python3 -m venv venv
source venv/bin/activate   # for linux
.\venv\Scripts\activate.bat  # for windows
pip install -e .

```
to restore the database :
```bash
flask db upgrade
```
